Welcome to WebViewToo

The WebViewToo class provides an easy to use method for building Web-based GUIs with AutoHotkey. It uses the same WebView2 framework from Microsoft present in Microsoft Edge.

This example is designed to show how you can use third party web frameworks like Bootstrap to build advanced user interfaces, while still keeping all the code local. Only minimal resources will need to be extracted from the compiled file.

As this example is more advanced, it assumes a stronger familiarity with the technology and may gloss over some parts more than other examples. If you're just getting started it may be helpful to work with some of the other example scripts first.
